# Full-Stack Blockchain App

## Deskripsi
Aplikasi web full-stack sederhana menggunakan React.js, Node.js, Express,
dan integrasi blockchain Ethereum (Sepolia) menggunakan Ethers.js.

## Fitur
- Connect wallet MetaMask
- Menampilkan address dan saldo ETH
- Mengambil data transaksi dari backend
- UI responsif menggunakan CSS Grid

## Teknologi
- React.js
- Node.js & Express.js
- Ethers.js
- Axios
- Ethereum Sepolia Testnet

## Cara Menjalankan
### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm start
